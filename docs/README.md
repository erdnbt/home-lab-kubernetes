# Operations Manuals

This folder contains practical manuals for the GitOps setup in this repository.

- `cluster-setup-manual.md`: how to bootstrap and validate the cluster + Flux.
- `application-deployment-manual.md`: how to organize and deploy applications with base/overlay structure.
- `monitoring-stack-manual.md`: how the monitoring stack is structured and operated.
- `sops-age-secrets-manual.md`: how to manage encrypted Kubernetes Secrets with SOPS + age.
